NOISE

聞き慣れた言葉。

ありふれた想い。

その器からこぼれた何かを、
観測する。

---

NOISEとは

声に出す前に、
まとまらず消えていく言葉。

繋がりかけては途絶える文脈。

意味になりかけて、
形を失う思考の断片。

NOISEは、それらを観測するためのアプリです。

意味を説明するものではありません。

意味を固定するものでもありません。

ただ、言葉がどのように見えるか——

その変化を観測します。

---

遊び方

1. NOISEを選ぶ（1枚）
2. 名詞・形容詞・動詞・世界観を選ぶ（最大2枚）
3. 「観測開始」を押す
4. 観測ログが生成される

観測ログは保存されません。

気に入ったログは、Xへ投稿できます。

---

観測ログについて

生成される文章は、
詩でも、小説でもありません。

NOISEは「観測ログ」と呼びます。

意味が成立しそうな場所で、
少しだけ形を崩す。

理解できそうで、
理解しきれない。

そんな断片を記録します。

例：

『約束』

透明な――
零れた

亡霊を――
少女

砂漠――
星雲

潜る――
漂う

---

NOISEという名前

ノイズは、邪魔なもの。

ノイズは、乱れたもの。

けれど時々、

そこには言葉にならなかった何かが
混ざっているのかもしれません。

---

注意事項

本アプリは意味の正解を提示しません。

生成結果の解釈は、
観測者に委ねられます。

良き観測を。
